豆种对比 · Species

Arabica 与 Robusta：两种截然不同的咖啡哲学

商业贸易中 99% 以上的咖啡来自这两个物种。Arabica 以风味取胜、需要更严苛的种植条件；Robusta 以高产、高咖啡因、抗病性著称，是速溶与意式拼配的支柱。

处理法 · Processing

从樱桃到生豆：五种主流处理工艺

处理法决定了豆子的发酵深度、干燥方式与最终风味。同一支咖啡换一种处理法，杯测分数与价格可以相差 5–10 美分以上。

价格体系 · Prices

三层定价：期货、ICO 综合价、与精品溢价

商业咖啡看 ICE 期货（纽约 Arabica、伦敦 Robusta），ICO 把各组别综合成 I-CIP 指数；产地端实际成交是 FOB / Farmgate 加减升贴水；精品市场则按杯测分数另行定价，可以远超期货。

查询思路 · How to Search

五种切片，把这份 Playbook 用起来

把咖啡问题转化为可比的查询维度。无论是给采购员还是给消费者，下面几条入口都能让信息更快落到决策上。

01

按风味

  • 明亮花果 → 埃塞 Yirgacheffe、肯尼亚 AA
  • 巧克力坚果 → 巴西 Santos、危地马拉 Antigua
  • 厚醇浓郁 → 印尼 Mandheling、印度 Robusta
02

按用途

  • 手冲 → 水洗高海拔 Arabica
  • 意式 / 拼配 → 巴西日晒 + 越南 / 印度 Robusta
  • 速溶 → 越南、印尼 Robusta
03

按价格段

  • 商业级：跟随 ICE 期货 + 升贴水
  • 精品级：杯测 84+，FOB 加 30–100% 溢价
  • 拍卖级：COE / Geisha，常突破 USD 30/lb
04

按产地集中度

  • 巴西 + 越南 ≈ 52% 全球供给
  • 前八国 ≈ 88%
  • 风险事件（霜冻、干旱）易传导价格
05

按处理法

  • 水洗 → 干净、酸度透亮
  • 日晒 → 果香、甜感、 body 厚
  • 蜜处理 / 厌氧 → 风味分层、精品偏好
